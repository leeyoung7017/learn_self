## 7.4. 分文件编写

### 7.4.1. 头文件

一般习惯导致这么写，灵活应用

```c++
#ifndef _FUNC_H

#include <iostream>
#include <string>

int add(int num1, int num2);

#endif // !_FUNC_H
#pragma once
```

### 7.4.2. 源文件

* 源文件是将函数写入，同函数的定义，但是不要忘记#include的命令
* 写完之后需要将头文件目录包含进去，网上查找即可；
